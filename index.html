⚡ Drop-in Plugin

Bitcoin Lightning Pay

A zero-dependency payment widget that connects directly to your LND node from the browser — no backend proxy required.

Configure Widget

Live Preview
Generated <script> tag 
<script src="https://lightning-pay.mariodian.com/lightning-pay.js"></script>

Required Params

macaroon req LND invoice macaroon — hex or base64/base64url
host req LND REST IP and port, e.g. 192.168.1.10:8080

Optional Params

amount Pre-set amount in satoshis. Hides the amount field.
expiry Invoice lifetime in minutes (default: 10)
showMessage true to show a memo/message field
hideQr true to hide the QR code
autoGenerate true to skip form and create invoice on load (requires fixed amount)

Invoice States

OPEN Waiting for payment. Progress bar counts down.
SETTLED Success message shown, confetti optional.
EXPIRED Error state with retry button.
🔑 Minimum Required Macaroon:
To use Lightning Pay, you need a macaroon with these permissions:
onchain:read address:read invoices:read address:write invoices:write
Generate it with the command below and use the resulting macaroon in the widget config above.
lncli bakemacaroon command 
lncli bakemacaroon onchain:read address:read invoices:read address:write invoices:write
⚠ CORS & TLS: Your LND node must allow cross-origin requests from the page's origin. Add --restcors=* (or the specific origin) to your LND startup flags, and ensure your node's REST API is accessible over HTTPS or from localhost. The macaroon is transmitted only to your own node — never to a third party.
